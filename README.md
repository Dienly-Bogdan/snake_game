# Snake Game

Простая реализация игры "Змейка" на Python с использованием Pygame.

## Как запустить

1. Установите зависимости:

    ```
    pip install -r requirements.txt
    ```

2. Запустите игру:

    ```
    python main.py
    ```

## Управление

- Стрелки — управление змейкой
- Пробел — рестарт после проигрыша

## Правила

- Ешьте красную еду, чтобы расти.
- Не врезайтесь в свое тело!
- После проигрыша нажмите пробел для рестарта.
